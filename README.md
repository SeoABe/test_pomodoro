# 포모도로 타이머 (메모리 상주형)

항상 위에 뜨는 작은 창 + 시스템 트레이 아이콘으로 동작하는 가벼운 포모도로 타이머입니다.
창을 닫아도 트레이에 남아 계속 동작하며, 트레이 아이콘에 남은 분이 숫자로 표시됩니다.

## 기능
- 집중 25분 → 짧은 휴식 5분, 집중 4회마다 긴 휴식 15분
- 단계 종료 후 **자동으로 다음 단계 시작** (설정에서 끌 수 있음)
- 트레이 아이콘에 **원형 진행률 게이지 + 남은 분** 표시 (단계별 색상)
- **설정 창**(⚙ 버튼 / 트레이 메뉴)에서 시간·옵션 변경, 자동 영구 저장
- 시작 / 일시정지 / 리셋 / 건너뛰기
- 단계 종료 시 트레이 풍선 알림 + 소리
- 창 닫기(X) → 트레이로 최소화, 트레이 메뉴에서 종료

## 설정 저장 위치
`%APPDATA%\pomodoro\config.json` 에 자동 저장됩니다. 설정 창에서 바꾸면 즉시 반영·저장됩니다.

## 바로 실행 (개발용)
```powershell
pip install -r requirements.txt
python pomodoro.py
```

## 배포용 단일 .exe 만들기
```powershell
.\build.bat
```
빌드가 끝나면 `dist\pomodoro.exe` 하나만 복사해서 어디서든 실행할 수 있습니다.
(파이썬 설치가 없는 PC에서도 동작)

### 시작프로그램 등록 (선택)
`Win + R` → `shell:startup` → 열린 폴더에 `dist\pomodoro.exe` 바로가기를 넣으면
부팅 시 자동으로 트레이에 상주합니다.

## 설정 변경
실행 중 **⚙ 설정** 버튼(또는 트레이 → 설정…)에서 집중/휴식 시간, 긴 휴식 주기,
자동 시작, 알림음을 바꿀 수 있습니다. 코드를 고칠 필요가 없습니다.
